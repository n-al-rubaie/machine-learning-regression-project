# Machine Learning Regression Project


This project demonstrates my ability to work with data, build regression models, and visualise predictions.

---

## Overview

- **Data:** `cw1data.csv`  
- **Tasks:** Data import, exploration, visualisation, train/test split  
- **Models:** Linear Regression, Random Forest Regression, Support Vector Regression (SVR), Gradient Boosting Regression  
- **Features:** Predictive modelling, comparison of model predictions, data visualisations  

---

## Key Skills

- Data analysis and cleaning with **Pandas**  
- Data visualisation using **Matplotlib** and **Seaborn**  
- Supervised machine learning regression techniques with **Scikit-learn**  
- Communicating results to technical and non-technical audiences  
- Handling numerical/tabular datasets for prediction tasks  

---

## Tools & Libraries

- **Python 3.x**  
- **Pandas** – data manipulation and cleaning  
- **Matplotlib** – plotting and visualisation  
- **Seaborn** – statistical data visualisation  
- **Scikit-learn** – machine learning models and train/test splitting
